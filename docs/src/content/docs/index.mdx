## Features

<CardGrid stagger>
	<Card title="Type-Safe" icon="approve-check">
		Comprehensive TypeScript types for all API operations. Catch errors at compile time, not runtime.
	</Card>
	<Card title="Modular Design" icon="setting">
		Use the unified client or install only what you need. Smaller bundle sizes for specialized applications.
	</Card>
	<Card title="Order Attribution" icon="puzzle">
		High-performance signing server built with Bun keeps your Builder credentials secure and server-side. [Learn more →](/order-attribution/overview)
	</Card>
	<Card title="Developer Experience" icon="star">
		Intuitive API design with detailed documentation, comprehensive examples, and helpful error messages.
	</Card>
</CardGrid>
